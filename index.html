Docker install nexus

Posted on 2019-05-04

Quick Start

  1. Search all nexus image in cloud
    docker search nexus3
  2. Pull one nexus3 image from cloud
    docker pull docker.io/sonatype/nexus3
  3. Check the local image
    docker images
  4. Start to run container
    docker run -id –privileged=true –name=nexus3 –restart=always -p 9003:8081 -v /kichun/nexus3/nexus-data:/var/nexus-data 6e9721ad473a（image Id that step3 get）

–privileged=true grant root permission for this container
–name=nexus3 container name “nexus3”
-p machine port：container port
-v machine path：container path mount a path
